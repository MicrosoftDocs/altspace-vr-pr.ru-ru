# <a name="using-the-groups-feature-to-create-private-events"></a>Создание частных событий с помощью функции "группы"

Наша функция групп дает возможность создавать пространство VR, открытое для утвержденного списка гостей. Узлы могут вручную добавлять имена пользователей и электронные сообщения Алтспацевр, которые им хотелось бы предоставить доступ к их пространству. Вот как можно использовать эту функцию для размещения событий, собраний и конференций:

> [!NOTE]
> Прежде чем получить доступ к "группам", необходимо включить функции раннего доступа, войдя в altvr.com, изменив профиль пользователя на веб-сайте и выбрав ранний доступ. или в приложении алтспацевр Параметры, как правило, участвуйте в программе раннего доступа.

1. В браузере перейдите к  [https://altvr.com](https://altvr.com)   своей учетной записи и войдите в нее. 
2. Найдите **раскрывающееся** меню.
3. Выбор **групп**

![Алтспацевр веб-представление с выделением дополнительных раскрывающихся меню](images/group-features-img-01.png)

4. На странице группы можно просмотреть список групп, в которые входит пользователь или администратор. 

![Страница "группы веб-представлений Алтспацевр"](images/group-features-img-02.png)

5. Нажмите кнопку **создать** , чтобы создать новую группу.
6. Заполните поля, чтобы создать группу.

![Страница создания новой группы с полями ввода](images/group-features-img-03.png)

> [!NOTE]
> Вы можете навести указатель мыши на зеленый символ "?" для получения дополнительных сведений о полях, имеющих их.

7. Отправьте сообщение электронной почты всем участникам и создайте учетную запись Алтспацевр, используя тот же адрес электронной почты. Они могут либо сделать учетную запись Алтспацевр в мире, либо на нашем [веб-сайте](https://account.altvr.com/users/sign_up).

## <a name="how-to-create-a-private-eventfrom-a-group"></a>Создание частного события из группы 

1. На странице элемент или администратор выберите группу, к которой требуется получить доступ. 
2. **Администратор** может изменить все параметры, настроенные ранее, для создания группы.Нажмите кнопку "Изменить" на странице группы, чтобы получить доступ к ним.  
3. **Добавление других администраторов для группы:**   При редактировании можно добавить других пользователей в качестве администраторов, чтобы упростить управление группой.  
    * Это должно быть имя пользователя Алтспацевр, а не электронная почта учетной записи.  
4. Выберите **Create Event (Создать событие)** .  

![Созданное событие](images/group-features-img-04.png)

## <a name="how-to-add-an-email-list-to-a-group"></a>Добавление списка сообщений электронной почты в группу 

Если у вас есть большая группа, которую необходимо добавить в группу, сначала необходимо создать список адресов электронной почты для всех пользователей, связанных с вашей группой. Помните, что если они подписываются с помощью связи с учетными записями Майкрософт, для выполнения этого шага потребуется свой адрес учетной записи Майкрософт.

### <a name="creating-a-list"></a>Создание списка

Используйте программу электронной таблицы, чтобы создать один столбец со всеми адресами электронной почты. После этого выполните следующие действия.  

1. Создание или изменение группы.  
2. Найдите поле **сообщения электронной почты для участников** .  
3. Найдите параметр **выбрать файл** , чтобы импортировать сообщения электронной почты из файла * .csv.
4. Список должен быть импортирован по одному адресу электронной почты в строке.

> [!NOTE]
> При выборе можно также скопировать и вставить сообщения электронной почты в поле, а не импортировать из файла * .csv.

### <a name="importing-a-list"></a>Импорт списка

Некоторые платформы могут позволять экспортировать файл * .csv (значение с разделителями-запятыми) с выделенным столбцом "Электронная почта".Этот столбец можно выбрать для использования.После того как этот доступ будет доступен, шаги в основном совпадают.  

1. Создание или изменение группы.  
2. Найдите поле **сообщения электронной почты для участников** .  
3. Найдите параметр **выбрать файл** , чтобы импортировать сообщения электронной почты из файла * .csv.
4. Список должен быть импортирован по одному адресу электронной почты в строке.

> [!NOTE]
> При выборе можно также скопировать и вставить сообщения электронной почты в поле, а не импортировать из файла * .csv.

### <a name="im-having-trouble-filling-out-the-fields-in-groups-what-do-they-mean"></a>У меня возникли проблемы при заполнении полей в группах. Что это означает?

(* Вы можете навести указатель на зеленую "?"для получения дополнительных сведений о полях, имеющих их.) 

* **Имя**   — Имя группы, которая будет видна всем элементам.   
* **Описание**   — Поле для описания назначения группы. Видимый для всех элементов.
* **Пользователи**   — Это поле позволяет перечислять членов группы для группы либо как имя пользователя, либо как адрес электронной почты, если он по одному на строку.Это один из способов определения членов группы.
    * Введите имя пользователя или адрес электронной почты в строке.  
    * Чтобы найти имя пользователя в Алтспацевр, пользователь должен открыть главное меню и перейти к разделу "Me".Это имя заключено в скобки.  
    * Чтобы найти имя пользователя на веб-сайте, пользователь должен зайти в "Profile" из параметра "дополнительные > Profile". Его можно найти в левом столбце страницы профиль.  
    * В этом поле учитывается регистр, поэтому имена "Username" и "Username" считаются разными.  
    * Это поле принимает как имена пользователей, так и сообщения электронной почты, связанные с действительными учетными записями.Убедитесь, что в строке есть один элемент.Вам не нужно вводить как сообщение электронной почты, так и имя пользователя, если они совпадают, либо будут работать.  
    * Учетные записи должны быть хорошо зафиксированы.  
    * Для поиска в списке можно использовать параметр поиска в браузере "Ctrl + F".  
* **Изображение**   — Используется как изображение рекламного баннера для группы.Он отображается на веб-странице группы и в VR.  
    * Мы рекомендуем 1920 x1080 для повышения точности изображения.  
    * Предпочтительный JPEG или PNG.  
    * Изображения должны составлять 16:9 пропорций.  
    * Мы автоматически создаваем эскизы из отправки и используем их в различных экранах пользовательского интерфейса в Алтспацевр и на веб-сайте.  
* **Изображение профиля**   — Используется как изображение профиля групп. Он отображается на веб-странице группы и в VR.  
    * Для повышения точности изображения рекомендуется 512 × 512.  
    * Предпочтительный JPEG или PNG.  
    * Мы автоматически создаваем эскизы из отправки и используем их в различных экранах пользовательского интерфейса в Алтспацевр и на веб-сайте.  
* **Выберите окружение**   — Располагается в правой части страницы, вы увидите различные эскизы доступных сред для использования. Эта среда будет действовать в качестве основного хангаут мира и будет отображаться в поле "мировое > мое WWN-> <имени из поля" имя "> Commons Attribution" для вас и членов вашей группы.  
    * Если вы хотите создать собственный мир, это можно сделать через [веб-сайт](https://account.altvr.com/worlds/my), и вы увидите эти сведения как возможные варианты. 

> [!NOTE]
> Эта функция доступна в бета-версии. Мы рекомендуем группам использовать существующие мировые области в диалоговом окне выбора среды.

* Слово «Commons Attribution» всегда будет отображаться как суффикс для любой группы хангаут мира.  
* **Сообщения электронной почты**   для участников — Это поле позволяет получить список пользователей по электронной почте, связанной с учетной записью Алтспацевр.  **Использование этого списка переопределяет список "Пользователи", описанный выше.**   Это рекомендуемый способ управления членством в группе.  
    * При заполнении этого поля убедитесь, что в нем содержится соответствующая электронная почта учетной записи Алтспацевр, или вы можете потерять доступ к группе. Если ваша учетная запись связана с компоновкой учетных записей Майкрософт, необходимо убедиться, что используемое электронное письмо является адресом вашей учетной записи Майкрософт.  
    * При вставлении адресов в поле убедитесь, что в строке есть по одному адресу электронной почты.  
    * Этот список принимает сообщения электронной почты без действительной учетной записи Алтспацевр.Имена пользователей Алтспацевр проверены, адреса электронной почты нет. Однако если это сообщение электронной почты используется для создания учетной записи Алтспацевр, то пользователь, с которым он связан, получит доступ к группе.  
    * Список можно импортировать из файла * .csv. Выберите параметр "выбрать файл" и выберите файл.
    * Мы рекомендуем использовать сообщения электронной почты как способ управления большими группами, так как это более полный способ согласования членов.Обратите внимание, что администраторам групп необходимо информировать пользователей и участников, что им необходимо использовать тот же точный адрес электронной почты, который использовался на платформе оплаты для создания учетной записи Алтспацевр.  
    * Сведения о том, как получить список пользователей, см. в разделе Импорт пользователей в группы.   
* **Включение автоматического импорта**   — Автоматический импорт — это повторяющийся процесс, выполняемый в списке разрешений электронной почты. Он согласовывает, имеет ли электронная почта связанную с ним учетную запись Алтспацевр и предоставляет доступ.   
    * Рекомендуется включить его, если вы используете отправку CSV-файла.
* **Идентификатор**   видео YouTube — Позволяет внедрить видео YouTube на целевую страницу группы на веб-сайте.Обратите внимание, что это только для веб-сайта группы и не отображается в Алтспацевр.
    * Необходимо использовать идентификатор видео, а не полный адрес. Идентификатор видео — это набор символов, отображаемых после входа в систему YouTube "=" (равно).  
* **Маркер Twitter**   — Внедрите последние твиты из учетной записи Twitter на целевую страницу вашей группы на веб-сайте.Обратите внимание, что это только для веб-сайта группы и не отображается в Алтспацевр.
* **Создать группу**   — Кнопка в нижней части страницы. Завершив поля, описанные выше, нажмите эту кнопку, чтобы создать группу.

1. Выберите элемент или администратор в верхней панели и выберите группу для управления или обновления. 
2. На странице элемент или администратор выберите группу, к которой нужно получить доступ, щелкнув ее изображение плитки.
